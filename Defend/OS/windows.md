# Windows

## User management

???+ tip "Manage WSUS state"
    | Command                                                   | Description |
    | --------------------------------------------------------- | ----------- |
    | `Set-Service -Name wsusservice -Status Running -PassThru` | Enable WSUS |
    | `Start-Service wsusservice  -PassThru`                    | Start WSUS  |
